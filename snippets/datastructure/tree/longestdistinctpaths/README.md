
### Description

Given a Tree, find count of distinct nodes in a root to leaf path with maximum distinct nodes.

### Example 1:

```
Input :   
          1
        /    \
       2      3
      / \    / \
     4   5  6   3
             \   \
              8   9 

Output : 4 

The root to leaf path with maximum distinct
nodes is 1-3-6-8.
```
